About

The source code for Oxytone is available on GitHub. It uses data from the PerseusDL project. It can also generate flashcards for use with the Anki software.

Oxytone uses color-coding to represent various aspects of Ancient Greek grammar:

Accusative
Dative
Genitive
Vocative
